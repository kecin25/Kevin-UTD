To run the bitmap project open Mars,
	go to Tools -> Bitmap Display
			set pixal dim to 4x4
			set display dim to 256x256
			Connect to MIPS
	go to Tools -> Keyboard and Display MMIO Simulator
			Connect to MIPS
			w - up
			a - left
			s - down
			d - right
			space - end game
	Run Mips		
		Goal of the game is to move the red pixal to the end point that is marked as a green pixel
